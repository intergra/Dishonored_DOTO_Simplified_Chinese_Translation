## 简介

本项目为《Dishonored: Death of the Outsider / 耻辱：界外魔之死》制作简体中文翻译，
覆盖菜单、HUD、任务、教程、技能与物品说明、书信、日志、环境文本、剧情对白和字幕。

翻译以官方英文原文为最高语义依据，并参考官方繁体中文。项目不是简单机械繁转简。
当前已经复用《耻辱2》中英文原文完全一致且完成审校的共有文本；DOTO 新增内容仍在
逐条检查说话场景、港台用语、误译、漏译、术语统一和表达自然度。

目前的数据已经完整枚举，但尚不能把全部 DOTO 专属初始种子称为完成审校的正式译文：

- 主文本 8,261 条：6,227 条复用已审校译文，2,034 条为 DOTO 专属待审校种子。
- 对白 5,431 条：2,363 条复用已审校译文，3,068 条为 DOTO 专属待审校种子。

## 兼容性与安全

当前基线为 Windows x64 Steam 版：

```text
Dishonored_DO.exe 1.145.0.0
```

项目使用 DOTO 独立的 Void Installer 工作目录，不与《耻辱2》的管理器状态混用。
构建从 DOTO 官方资源只读提取语言、字体和对白模板，并在一次性沙箱中验证首次安装；
构建过程不会修改真实游戏目录。

本项目不会复制《耻辱2》的原生运行时启动器、注入 DLL、EXE 偏移或对象布局。

当前静态安全底座只打包主语言表、字体和 17 个必须直接替换的对白资源；其余 905 个
声明保持原版加载生命周期。完整人工译文由 DOTO 专用轻量运行时字幕层负责：只在原版
声明加载完成后修改既有字幕字符串，不重建声明，不接管地图、存档、检查点、角色或
音频生命周期，因此每一个文本槽都可以独立精修，不受官方繁体字数和语序限制。

DOTO 专用运行时目录、启动器和 DLL 已完成 `v1.0` 完整包构建、独立回读和首次安装
沙箱验证。公开 Release 仍未创建；正式公开前还需要作者按普通玩家流程从 Steam 启动，
完成主菜单、实际关卡、检查点、死亡回档、地图切换和返回主菜单等实机验收，并继续精审
DOTO 专属译文。实机验证失败时必须停止发布并修正问题。

## 预计安装方式

正式版本发布后：

1. 从本仓库的 Releases 页面下载单独提供的 MOD ZIP。
2. 把 ZIP 解压，取得第一层含 `ModInfo.xml`、`Resources` 和 `Runtime` 的 MOD 文件夹。
3. 使用 Void Installer，Game folder 选择包含 `Dishonored_DO.exe` 和 `base` 的游戏目录。
4. Mod folder 选择含 `ModInfo.xml` 的 MOD 文件夹并安装。
5. 在 Steam 中把游戏语言设置为中文（繁体）。
6. 打开 Steam 的“属性 → 通用 → 启动选项”，填写下方一整行命令，把路径替换为自己的
   MOD 解压目录：

```text
"你的MOD目录\Runtime\DOTORuntimeLauncher.exe" --steam-command %command%
```

7. 以后仍然只从 Steam 的“开始游戏”按钮启动。无需也不建议到游戏目录双击
   `Dishonored_DO.exe`。

启动器会自动读取同目录的 `DOTORuntimeTranslation.dll` 和
`speech-translations.bin`，校验游戏版本后把 Steam 传入的真实 `%command%` 原样交给
游戏。它完成翻译 DLL 的就绪确认、恢复游戏主线程后就会立即退出，所以启动时看到窗口
一闪而过属于正常现象；翻译 DLL 会留在游戏进程中继续工作。解压目录需要长期保留；
移动目录后必须同步更新 Steam 启动选项。

开发期间可选的外部只读检查脚本不属于玩家安装内容，也不会进入 Release ZIP。玩家包
只有一次完整安装和上面这一条 Steam 启动命令，不需要额外测试包或第二条启动命令。

正式发布前，具体文件名、版本号和 Release 链接不会提前写死。

## 翻译原则

1. 英文原文是判断含义的最高依据。
2. 结合剧情流程、人物身份、组织性质和说话场景。
3. 使用符合中国大陆玩家习惯和游戏行业约定的表达。
4. 人物、地点、组织、技能和物品译名保持一致。
5. UI、任务与教程优先保证清晰准确。
6. 对白、书信和日志保持系列冷峻、克制的工业奇幻气质。
7. 保留变量、占位符、标签、转义和所有技术控制结构。
